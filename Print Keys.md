# Print Object Keys

Fork this repository and solve the following problem.
---

You are given an object with key-value pairs. Your task is to write a function called printKeys that will:

Extract all the keys from the object.  
Print each key on a separate line.

## Input
For user-generated test cases:
1. The first line of input will contain the number of field the object should have, i.e. n.
2. Next n line contains : separated key-value pairs.

Note: You are not required to take any input. You will be given the object, you only have to print the keys.

## Output
You are required to print all the keys present in the object, one in each line.
