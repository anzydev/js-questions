# Multiply All Elements of an Array

Fork this repository and solve the following problem.
---

You are given an array of numbers. Your task is to finish a JavaScript function that will multiply all the numbers in that array and return the final result.

NOTE: If the array is empty, return 0.

Note: This is a functional problem; you do not have to handle the input or output. Complete the function and return the value.

## Input
A single line containing an array of numbers.

## Output
A single number: the product of all the elements in the input array.
