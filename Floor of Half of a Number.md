# Floor of Half of a Number

Fork this repository and solve the following problem.
---

You are given an integer n as input. Your task is to:  
First, calculate half of the given number.  
Then, print the floor value of that half.  

The floor value is the greatest integer that is less than or equal to a given number.

## Input
The input consists of a single integer n.

## Output
Print the floor value of n / 2.
