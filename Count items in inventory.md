# Count Inventory Items

Fork this repository and solve the following problem.
---

You are given a JavaScript object, stored in a variable "inventory" where:  
Each key represents the name of an item (e.g., "pen", "notebook")  
Each value represents the quantity of that item.

Your task is to calculate how many different items are present in the inventory. Print the result in the following format  
Total items: X

If the object is empty, the output should be  
Total items: 0

## Input
A single line containing a JSON Stringified object.

## Output
A string "Total items: X", where X is the number of different items present in an object "inventory"
